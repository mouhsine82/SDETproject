Method overloading:
   It's a mechanism in java that allows us to have same method name with different parameters.
   Parameters can be different in number, type or sequence.
Why we should overload a method?
    Method overloading allows us t reuse same method name again and again that we don't have to come up with different
    method names if we are doing the same thing again and again.
How to overload a method:
    We can overload a method be redefining it with different parameters.

Method overriding:
   It's a mechanism in java that allows a subclass or child class to provide a specific implementation of a method.
Why we should override a method?
   Is a child class doesn't like the implementation of the parent class it has the flexibility that you can override that method.
Syntax of override?
  There must be a parent child relationship between two classes.
